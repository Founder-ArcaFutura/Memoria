Editing tables from the dashboard

Signed-in operators with the Memoria admin API key (or a valid UI session token) can now add, edit, and delete rows directly from the table browser.

  • Add row opens a form generated from column metadata, highlighting required fields and applying basic type hints before the request is sent.
  • Edit and Delete controls are exposed per row when a primary key is available. The dashboard performs client-side validation, surfaces API errors, and keeps the current namespace/search filters after a successful mutation.
